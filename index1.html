Relatório do Experimento: Simulação de Semáforo com Arduino

1. Introdução

Este experimento teve como objetivo simular o funcionamento de um semáforo utilizando um microcontrolador Arduino, três LEDs (vermelho, amarelo e verde) e uma protoboard. O semáforo é um dispositivo utilizado para controlar o fluxo de veículos e pedestres em cruzamentos. No experimento, replicamos a sequência de mudança de luzes para observar o comportamento do sistema e aplicar conceitos de eletrônica e programação com Arduino.

2. Materiais Utilizados

  • 1 Arduino (UNO ou similar)
  • 1 Protoboard
  • 1 LED vermelho
  • 1 LED amarelo
  • 1 LED verde
  • 3 resistores de 220Ω
  • Fios jumpers para conexões
  • Cabo USB para alimentação e programação do Arduino
  • Software Arduino IDE

3. Esquema de Montagem

A montagem foi realizada conectando os LEDs à protoboard e ligando-os às portas digitais do Arduino conforme o seguinte esquema:

  • LED Vermelho: conectado à porta digital 9 do Arduino.
  • LED Amarelo: conectado à porta digital 10.
  • LED Verde: conectado à porta digital 11.

Cada LED foi conectado em série com um resistor de 220Ω para limitar a corrente, evitando queima dos LEDs. O pino GND do Arduino foi conectado à linha negativa (terra) da protoboard, e todos os cátodos dos LEDs foram ligados à essa linha.

4. Código Fonte


// Definindo as portas dos LEDs
int ledVermelho = 9;
int ledAmarelo = 10;
int ledVerde = 11;

void setup() {
  // Configurando os pinos como saídas
  pinMode(ledVermelho, OUTPUT);
  pinMode(ledAmarelo, OUTPUT);
  pinMode(ledVerde, OUTPUT);
}

void loop() {
  // Luz verde acesa (simulando trânsito liberado)
  digitalWrite(ledVerde, HIGH);
  delay(5000);  // Mantém por 5 segundos
  digitalWrite(ledVerde, LOW);

  // Luz amarela acesa (simulando atenção)
  digitalWrite(ledAmarelo, HIGH);
  delay(2000);  // Mantém por 2 segundos
  digitalWrite(ledAmarelo, LOW);

  // Luz vermelha acesa (simulando parada)
  digitalWrite(ledVermelho, HIGH);
  delay(5000);  // Mantém por 5 segundos
  digitalWrite(ledVermelho, LOW);
}

5. Funcionamento

Ao carregar o código no Arduino, observou-se a seguinte sequência:

  1. O LED verde acende por 5 segundos, representando a liberação do tráfego.
  2. O LED amarelo acende por 2 segundos, alertando sobre a mudança iminente para a luz vermelha.
  3. O LED vermelho acende por 5 segundos, indicando que o tráfego deve parar.

A sequência se repete indefinidamente, simulando o comportamento de um semáforo tradicional.

Montagem Final

6. Análise dos Resultados

O experimento foi bem-sucedido na simulação do semáforo. A troca de estados entre os LEDs ocorreu conforme esperado, e o código foi eficiente para controlar as luzes de acordo com o tempo configurado. O uso dos resistores garantiu a proteção dos LEDs e evitou danos ao circuito.

7. Conclusão

Este experimento demonstrou o funcionamento básico de um semáforo utilizando componentes eletrônicos simples e programação com Arduino. A partir da simulação, foi possível compreender melhor o controle de dispositivos através de código e o conceito de temporização, essenciais em projetos de automação.

8. Possíveis Melhorias

Algumas melhorias poderiam ser implementadas, como a inclusão de um botão para simular a intervenção de pedestres no sistema ou a implementação de um sistema mais complexo de semáforos para cruzamentos.
